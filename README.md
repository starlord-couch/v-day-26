# v-day-26
Valentine Compatibility, Invitation Python Script

This script launches a terminal window where the user's 'Valentine Compatibility' is verified.  The user is prompted to answer three questions which have no bearing on the functionality of the rest of the script.  The user is confirmed compatible then prompted to answer if they wish to be your valentine.  Answering 'Yes' confirms with a final line to end the terminal session.  'No' returns the user back to the valentine confirmation question.
